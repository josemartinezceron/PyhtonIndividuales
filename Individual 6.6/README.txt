##Descripción 
Esta aplicación web está diseñada para vender contenido exclusivo a usuarios registrados. Los usuarios deben pagar para acceder al contenido y se les otorgará un permiso especial de Django una vez que se hayan registrado y pagado. 
 
## Permisos de Django 
En este caso, la aplicación utiliza un permiso simple de Django para restringir el acceso al contenido solo a usuarios registrados que hayan pagado. Los usuarios que no hayan pagado no podrán acceder al contenido exclusivo.  
 
## Instalación 
1. Clonar este repositorio en tu máquina local. 
2. Crear un entorno virtual y activarlo. 
3. Instalar los requerimientos del archivo requirements.txt. 
4. Ejecutar las migraciones de Django para crear la base de datos. 
5. Ejecutar la aplicación web. 
 
## Uso 
1. Regístrate como usuario en la aplicación. 
2. Realiza el pago para acceder al contenido exclusivo. 
3. Una vez que hayas pagado, se te otorgará el permiso especial de Django para acceder al contenido exclusivo. 
4. Disfruta del contenido exclusivo.
